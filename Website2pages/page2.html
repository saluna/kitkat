My Most Favorite Ted Talk I've Found On Youtube

The birds make flying seem so easy. But trying to replicate that? Eh, not so much. The motors, sensors, metal, etc. that is needed to build a robot all add up fairly quickly, which usually prevents flight. This video, however, demonstrates that one team was able to accomplish the feat that involved many calculations and each piece of the robot had to be taken into consideration. Though it doesn't look like it due to the simply body, something so small can be stunningly complicated yet beautiful.
